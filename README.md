# Caso Práctico — Técnicas de Producción Industrial de Software II

## Descripción
Análisis de datos aplicado al dataset Prestige_modificado.csv en el contexto 
de la empresa ficticia InnovaSoft Analytics. El objetivo es identificar 
patrones en ocupaciones para apoyar decisiones sobre talento y perfiles 
profesionales.

## Estructura del repositorio
- DATA/ — Dataset utilizado en el análisis
- Notebook/ — Notebook de Google Colab con el análisis completo

## Actividades desarrolladas
1. Preparación e inspección del dataset
2. Análisis de correlación
3. Visualización de resultados
4. Clusterización K-Means (4 y 5 grupos)
5. Interpretación orientada a producción de software

## Tecnologías utilizadas
- Python 3
- pandas, numpy
- matplotlib, seaborn
- scikit-learn
